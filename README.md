# PATROL - Nissan Patrol Specialist Garage

🚗 **Only Nissan Patrol Vehicle Repairing**  
A modern, bilingual (Arabic/English) website for a specialized garage in Doha, Qatar.

## Features
- 🌐 **Bilingual** - Arabic (default) and English toggle
- 📱 **Responsive** - Works on all devices
- 🎨 **Modern Design** - Dark theme with orange accents
- 📧 **Contact Form** - Sends messages to patrolgarageqa@gmail.com via Formspree
- 💬 **WhatsApp Integration** - Direct chat button
- 🔄 **Smooth Animations** - Scroll reveals and hover effects

## Technologies
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Formspree (form handling)
- Font Awesome Icons

## Deployment
Deployed on Vercel.
